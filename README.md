Welcome to the static landing page for **AI-Proctor**—a secure, AI-powered online examination and intelligent proctoring system. This repository contains the fully responsive, statically hosted webpages for the project landing, login, registration, FAQ, and contact sections.

---

> [!IMPORTANT]
> This repository is a **fully static frontend client**. It is designed to act as the public-facing landing and product informational site. It does not run the backend Python models or database checks locally, making it perfect for free, high-performance static hosting on **GitHub Pages**, **Netlify**, or **Vercel**.

👉 **Live Site Link:** [https://Aditya1791.github.io/ai-proctor-landing/](https://Aditya1791.github.io/ai-proctor-landing/)

---

## 🚀 Key Aesthetics & Styling Highlights
* **🌌 Premium Cyber-Neon Glow:** Features custom color gradients, text-shadow drops, and glowing button frames powered by a custom design system.
* **🧪 Frosted Glassmorphism:** Cards and navigations built with `backdrop-filter: blur` and subtle translucent borders (`rgba` transparency).
* **✨ Smooth Micro-Animations:** Animate-on-Scroll (AOS) transitions, VanillaTilt 3D tilt effects, and Lottie-player vector animations.
* **🖥️ Custom Cyber Scrollbar:** Scrollbars dynamically styled with a neon-cyan to neon-purple gradient.

## 🛠️ Local Development & Quickstart
You do not need any installation or compiler. Simply open the files directly:

1. **Clone the Repo:**
   ```bash
   git clone https://github.com/Aditya1791/ai-proctor-landing.git
   ```
2. **Launch the page:**
   * Double-click **`index.html`** to run locally in any browser.
   * *Or* run a light local web server:
     ```bash
     # Python 3
     python -m http.server 8000
     ```
     Visit `http://localhost:8000` in your web browser.
