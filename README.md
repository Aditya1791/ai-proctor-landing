# AI-Proctor Landing Page 🎓🤖

Welcome to the static landing page for **AI-Proctor**—a secure, AI-powered online examination and intelligent proctoring system. This repository contains the fully responsive, statically hosted webpages for the project landing, login, registration, FAQ, and contact sections.

## 🚀 Live Demo & Hosting
This static site is optimized to be hosted on **GitHub Pages**, **Netlify**, or **Vercel** with zero backend requirements. 

If hosted on GitHub Pages, the site is accessible at:
👉 **[https://Aditya1791.github.io/ai-proctor-landing/](https://Aditya1791.github.io/ai-proctor-landing/)**

---

## 👁️ Features
* **Futuristic Cyber-Neon Design:** Frosted glassmorphism cards, glowing text gradients, custom CSS variables, and animated elements.
* **Responsive Layout:** Powered by Bootstrap 4.3.1 to ensure a clean, mobile-first presentation.
* **Interactive Elements:** Uses Lottie animations, SweetAlert2 popups, VanillaTilt card tilts, and AOS (Animate on Scroll) transitions.
* **No-Backend Static Forms:** Standardized placeholders for Login, Register, FAQ, and Contact pages, fully routed through relative URLs.

## 🛠️ How to Run Locally
1. Clone this repository to your local machine:
   ```bash
   git clone https://github.com/Aditya1791/ai-proctor-landing.git
   ```
2. Double-click **`index.html`** to open it directly in your web browser.
3. Alternatively, run a simple local development server:
   * **Python 3:** `python -m http.server 8000` (Access at `http://localhost:8000`)
   * **VS Code:** Install the **Live Server** extension and click **Go Live**.
